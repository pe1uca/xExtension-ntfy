# Extension to integrate FreshRSS with ntfy.sh  

This extesion allows you to configure a ntfy.sh server and topics to send simple notifications about new articles from your feeds.  

The default topic receives one notification per feed.  

# Configuration  

Once the extension has been installed and enabled, you can:  

- Configure the ntfy.sh server to use.  
- Configure a topic to send the notifications.  
- Configure a topic per feed.
